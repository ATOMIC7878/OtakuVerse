## 🌟 Overview

OtakuVerse is a production-ready full-stack notes sharing platform built with modern web technologies. It provides a futuristic, anime-inspired learning environment where users can access, download, and manage study notes across multiple domains.

## 🚀 Features

### User Features

- **Authentication System**: Secure JWT-based authentication with bcrypt encryption
- **Domain Categories**: CS, DS, and IT domains with unique themes
- **Notes Management**: View, download, and preview study materials
- **Responsive Design**: Perfect experience on all devices
- **Modern UI**: Glassmorphism effects, video backgrounds, smooth animations

### Admin Features

- **Isolated Admin Panel**: Completely separate from user interface
- **Content Management**: Upload, edit, and delete notes
- **User Management**: View and manage user accounts
- **Analytics Dashboard**: Track platform statistics

### Technical Features

- **Security**: Helmet.js, CORS, rate limiting, XSS protection
- **Database**: MongoDB with Mongoose ODM
- **API**: RESTful API architecture
- **Animations**: Three.js particle system, Anime.js transitions
- **Video Background**: Looping background video with overlay
